# Portfolio, Halvor Nilsen Stensby.

Dette er den offisielle github-repoen for Halvors portfølie. Utviklet selv av halvor.

## Innhold 

Denne portefølje-nettsiden er bygget for å gi en oversiktlig presentasjon av bakgrunn, ferdigheter og arbeid. Strukturen er delt inn i tydelige seksjoner for å gjøre informasjonen lett tilgjengelig både for besøkende og for videre utvikling av prosjektet.

Nettsiden inneholder:

    Header – Navn og kort beskrivelse som introduserer siden

    Om meg – En personlig introduksjon med profilbilde og bakgrunn

    Prosjekter – Utvalgte arbeider med beskrivelser av hva som er laget og hvilke teknologier som er brukt

    Ferdigheter – Oversikt over relevante tekniske og praktiske kompetanser

    Kontakt – Informasjon for å komme i kontakt via e-post eller profesjonelle plattformer

    Footer – Enkel bunntekst med opphavsrett

Strukturen er laget med semantisk HTML for å sikre god lesbarhet, tilgjengelighet og et solid grunnlag for videre styling og funksjonalitet.
